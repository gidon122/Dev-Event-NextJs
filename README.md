# 🚀 DevEvent

> **Discover. Learn. Build. Connect.**

DevEvent is a modern developer event discovery platform built with **Next.js** that helps developers discover hackathons, conferences, meetups, workshops, bootcamps, webinars, and other tech events happening around them.

Whether you're a student, software engineer, designer, founder, or tech enthusiast, DevEvent makes it easy to find events that help you learn new skills, network with professionals, and grow your career.

---

## 🌍 Vision

Technology grows through communities.

DevEvent aims to become the central hub where developers can:

- Discover upcoming tech events
- Register for events
- Connect with local communities
- Explore past events
- View event photos and reviews
- Learn from industry experts
- Grow their careers through networking

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/devevent.git
```

Navigate into the project

```bash
cd devevent
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```

---

# 📦 Environment Variables

Create a `.env.local` file.

Example:

```env
NEXT_PUBLIC_POSTHOG_KEY=your_posthog_key
NEXT_PUBLIC_POSTHOG_HOST=https://app.posthog.com
```
